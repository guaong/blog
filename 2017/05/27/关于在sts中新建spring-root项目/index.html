关于在sts中新建spring-root项目

超级无敌操蛋的sts，啊啊啊啊啊！！！！！！！！！！


经过一下午琢磨，终于，终于还是没有完成，不过也解决啦一部分问题

根据spring官网给出的文档
第一步
检测是否安装jdk
$ java -version

第二步
maven安装(直接从网上下载),下载之后解压

第三步
配置环境变量
path里面添加%M2_HOME%\bin
新建M2_HOME,添加E:\apache-maven-3.5.0

第四步
控制台输入$ mvn -v检测是否安装成功

第五步
新建项目spring starter project

第六步
之后将pom.xml文件与官网给的doc比较，大多数都自动添加了,只有:

< dependencies>
< dependency>
< groupId> org.springframework.boot </ groupId>
< artifactId> spring-boot-starter-web </ artifactId>
</ dependency>
</ dependencies>没有添加,当添加之后会下载一些东西,我猜大概是org.spring.web包,因为之前没有该报,但是添加这段之后运行程序出现异常，大致是有重复包，导致项目依赖的包与拥有的包不符,之前没有添加这段，程序一直是一运行就停止

最后
运行，至少根据以上步骤，服务器可以一直启动，不会自动停止

话外
其实maven可以改变，设置window首选项,将maven中useinterface中的文件位置改变,粗略一提，原本文件在user/.m2/respxxx下，将位置改变，因为该文件下将下载jar包.


未完待续，我将继续解决网页404问题…

最后最后一步
将@SpringBootApplication改为@RestController和@EnableAutoConfiguration然后运行就没有问题了


至此简单的新建spring boot并运行就完成了!
