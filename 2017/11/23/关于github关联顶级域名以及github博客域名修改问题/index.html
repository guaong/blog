关于github关联顶级域名以及github博客域名修改问题

关于github关联顶级域名的说明



我个人使用的顶级域名是.top，其他域名类似
1.购买域名
2.买好域名后，要进行解析，以阿里云为例，找到解析设置，添加解析
3.添加两条，分别是主机记录为www和@
4.记录值填写你的github的ip，在cmd中ping user.github.io,得到ip
5.填写完成后确认即可(之前百度上有很多老的教程，ip都是192开头的，我亲自操作发现并不能实现，另一种是直接填写user.github.io，我没有测试)
6.由于我使用的是.top因此不需要备案，也不需要实名认证
7.然后此时开始修改hexo
8.在本地hexo中新建CNAME文件，没有扩展名，在CNAME中填入你买的域名，保存，然后hexo g hexo d
9.此时在网页的github中你博客那个仓库中，点settings向下拉，发现你的href变成了你的域名
10.这样就完成啦


关于博客域名修改问题，仅针对hexo



1.在github网页中找到你的博客仓库，点击settings，修改名字
2.在本地的hexo中的_config文件中修改url,root,deploy中的repository,至于怎么改一看就知道了
3.hexo g hexo d就ok了
