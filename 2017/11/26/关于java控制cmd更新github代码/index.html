java控制cmd


有两种方式：
1.直接运行批处理文件
2.运行cmd命令


由于第二种没有找到合适的可以连续执行cmd命令的方式，所以只对第一种进行讲解
1.首先新建一个批处理文件
2.在里面写入指令(一个指令一行)
例如：

C:
cd \Users\关桐\Documents\GitHub\home
git status
git add *
git commit -m 00
git pull
git push origin master

3.值得注意的是跳转磁盘不能用cd，而打开文件要用cd，以上就是标题的完整代码
4.还有运行bat的代码,这是两种方式，第一种是会弹出黑框，第二种不会

Desktop.getDesktop().open(new File(“C:\Users\关桐\Documents\GitHub\post.bat”));
final Process r = Runtime.getRuntime().exec(“C:/Users/关桐/Documents/GitHub/post.bat”);
